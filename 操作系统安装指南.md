
# debian操作系统自动化安装向导

* **安装操作系统** 
1、确认设备硬盘类型（硬盘 or Reed），选择安装光盘
2、插入U盘，连接显示器、键盘、鼠标、网线（Lan B）
3、设备上电，不断按del键，进入bios模式
4、设置Boot优先级，Boot Option #1，选择UEFI：Mass Storage Device 1.00，Partation1
5、Save and Exit,点击Yes
6、自动安装操作系统，完成会自动重启

* **安装基础软件库**
登录root账号，用户:aiclab.org,密码：aiclab.org
执行./publish/install.sh，安装基础软件库

* **确认安装完毕**
ip a，查看设备IP（enpos31f6）
打开浏览器，访问IP，看到RTFrame即表示安装成功
